MINI SIEM DASHBOARD

How It Works

Mini SIEM Dashboard simplifies security monitoring. Users export raw security logs from Windows Event Viewer as a CSV file and upload them to the dashboard. The system instantly analyzes the logs, automatically calculates the severity of potential threats (like repeated failed logins), and visualizes the data through interactive charts and tables for quick threat assessment.
